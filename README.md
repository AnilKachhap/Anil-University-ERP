\# 🎓 Anil University ERP Management System



Anil University ERP is a web-based Enterprise Resource Planning system developed to manage university operations digitally. The system provides modules for student management, course management, attendance tracking, notifications, payments and student portal.



\## 🚀 Features



\### Admin Module

\- Admin login and authentication

\- Dashboard with statistics

\- Student management

\- Course management

\- Attendance management

\- Payment management

\- Notification management



\### Student Module

\- Student registration

\- Profile management

\- Document upload

\- Attendance tracking

\- Payment history

\- Student ID Card generation



\## 🛠️ Technologies Used



\- ASP.NET Core MVC

\- C#

\- Entity Framework Core

\- SQL Server

\- ASP.NET Core Identity

\- Bootstrap 5

\- jQuery

\- HTML5

\- CSS3

\- JavaScript



\## 🏗️ Architecture



\- MVC Design Pattern

\- Repository Pattern

\- Service Layer

\- Dependency Injection

\- Entity Framework Core Code First Approach



\## 💳 Payment Integration



\- Razorpay Payment Gateway Integration

\- Online payment processing

\- Payment receipt generation
